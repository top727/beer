# beer

## Overview
This is the Shane's beer test project

### backend

#### technique
- Express JS (Api REST)
- socket.io
- Axios
- Mocha, Chai (Testing)
- Babel (ES6 support)

#### Scripts

Here is the list of all scripts inside, you have to go to **borderline** folder to execute them.

- Go to backend project
    - >cd backend
- Run to start application
	- >npm start

### frontend

#### technique
- React.js
- socket.io-client
- SCSS
- Jest(Testing)
- Babel (ES6 support)
- Webpack

#### Scripts

Here is the list of all scripts inside, you have to go to **borderline** folder to execute them.

- Go to frontend project
    - >cd frontend
- Run to start application
	- >npm start
- Run the tests
	- >npm test
